أريد إنشاء موقع ويب تعريفي احترافي لصيدلية باسم:

صيدلية د/ محمد الفرا - El-Farra Pharmacy

الموقع سيكون تعريفي فقط (ليس متجر إلكتروني، ولا يوجد منتجات أو نظام شراء).

⚙️ التقنيات المطلوبة:
- React (باستخدام Vite)
- JavaScript فقط (بدون TypeScript)
- CSS بسيط أو Tailwind (يفضل تصميم خفيف وسهل التعديل)
- بدون Backend
- الكود يكون منظم وسهل التعديل
- تعليقات داخل الكود لشرح كل جزء

🌍 اللغات:
- الموقع يدعم العربية والإنجليزية
- يكون RTL للعربية
- زر لتبديل اللغة (Language Switcher)
- استخدام نظام بسيط لإدارة الترجمة (مثلاً ملف JSON يحتوي على النصوص)

- سهولة الاستخدام أولوية
- أنيميشن خفيف عند التمرير (Fade In بسيط)
